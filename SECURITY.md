<p align="center">

| Versão | Suporte |
|--------|---------|
| 1.0.x  | ✅      |

---

## 🚨 Reportar Vulnerabilidades

Se você encontrar alguma vulnerabilidade no **Sistema de Tarefas API com integração com a API dos Correios**, siga os passos abaixo:

### 📩 Como reportar

- Abra uma **issue privada** no GitHub *(se disponível)*  
ou  
- Entre em contato diretamente com o desenvolvedor

### 📝 Informações importantes

Inclua o máximo de detalhes possível:

- Descrição da vulnerabilidade  
- Passo a passo para reprodução  
- Possível impacto  
- Evidências (prints, logs, etc.)

---

## ⏱️ Tempo de resposta

- 📬 Retorno inicial: até **3 dias úteis**
- 🔍 Análise completa: o mais rápido possível

---

## ✅ Se a vulnerabilidade for confirmada

- Correção será aplicada rapidamente  
- Nova versão segura será publicada  
- Créditos ao responsável (opcional)  

---

## ❌ Se não for confirmada

- Será enviada uma explicação detalhada do motivo  

---

## 🛡️ Boas práticas de segurança

Este projeto foi desenvolvido seguindo boas práticas:

- 🔐 Autenticação com JWT  
- ✅ Validação de dados de entrada  
- 🗄️ Uso de Entity Framework Core (proteção contra SQL Injection)  
- 🌐 Integração segura com API externa (Correios)
